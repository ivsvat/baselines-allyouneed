# baselines-allyouneed
The official repository for "Deep Learning for BioImaging: What Are We Really Learning?" (WIP)

## References to external evaluation

### RxRx3-core
We follow the evaluation setup from [Kraus et al., 2025](https://arxiv.org/abs/2503.20158) : [code](https://github.com/recursionpharma/EFAAR_benchmarking/tree/trunk/RxRx3-core_benchmarks), [data](https://huggingface.co/datasets/recursionpharma/rxrx3-core).

### JUMP-CP, 8 positive controls
We follow the evaluation setup from [Sanchez et al., (2026)](https://www.nature.com/articles/s42003-025-09500-y#code-availability): [code](https://github.com/mxfly14/2025_sanchez_phenoseeker).

### JUMP-MoA
We follow the evaluation setup from [Stossi et al., 2024](https://www.nature.com/articles/s41467-024-54264-4) : [data](https://github.com/jump-cellpainting/JUMP-MOA)

### HEST-1k (non-binned)
We follow the evaluation setup from [Jaume et al., (2024)](https://dl.acm.org/doi/10.5555/3737916.3739620): [code](https://github.com/mahmoodlab/hest), [data](https://huggingface.co/datasets/MahmoodLab/hest)

### BRACS and BACH
We follow the evaluation setup from [Öğüt.,(2026)](https://arxiv.org/abs/2603.00143): [code](https://github.com/ogutsevda/graphist), [BRACS](https://www.bracs.icar.cnr.it/)
